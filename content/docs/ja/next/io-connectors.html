Builtin Connectors

Pulsar distribution includes a set of common connectors that have been packaged and tested with the rest of Apache Pulsar. These connectors import and export data from some of the most commonly used data systems. Using any these connectors is as easy as writing a simple connector configuration and running the connector locally or submitting the connector to a Pulsar Functions cluster.

  • Aerospike Sink Connector
  • Cassandra Sink Connector
  • Kafka Sink Connector
  • Kafka Source Connector
  • Kinesis Sink Connector
  • RabbitMQ Source Connector
  • Twitter Firehose Source Connector
