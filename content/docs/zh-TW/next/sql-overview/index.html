Pulsar SQL 概述

Apache Pulsar 用於存儲事件數據流，而事件數據是以預定義的欄位結構來保存、傳送與應用。 使用 Schema Registry， 你可以在 Pulsar 中存儲結構化數據，並通過使用 Presto 來查詢數據。

Pulsar SQL 的核心是 Presto Pulsar 連接器，它允許 Presto worker 從 Pulsar 查詢數據。

Pulsar SQL 查詢性能非常高效且可高度擴展的原因是 Pulsar 的分層分片資料存儲架構。

Pulsar 中的 topic 使用 segment 的概念將信息資料存儲到 Apache Bookkeeper 的 segment 中。 每個 topic 的 segment 都被複製到某些 BookKeeper 節點，這設計提高了同時讀和寫的 throughput。 你可以配置 BookKeeper 節點的數量，預設的數量為 3</ code>。 在 Presto Pulsar 連接器中，會直接從 BookKeeper 讀取數據，因此 Presto worker 可以從水平可擴展數量的 BookKeeper 節點上並發讀取。
